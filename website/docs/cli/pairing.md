# `openkrab pairing`

Approve or inspect DM pairing requests (for channels that support pairing).

Related:

- Pairing flow: [Pairing](/channels/pairing)

## Commands

```bash
openkrab pairing list whatsapp
openkrab pairing approve whatsapp <code> --notify
```
