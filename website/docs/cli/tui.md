# `openkrab tui`

Open the terminal UI connected to the Gateway.

Related:

- TUI guide: [TUI](/web/tui)

## Examples

```bash
openkrab tui
openkrab tui --url ws://127.0.0.1:18789 --token <token>
openkrab tui --session main --deliver
```
